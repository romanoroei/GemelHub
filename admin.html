חזרה לאתר

כניסה לניהול

הכנס token ניהולי כדי לקרוא נתונים מה-Worker. ה-token נשמר בדפדפן שלך בלבד.

Worker ממתין לבדיקה
KV שיתופים ממתין לנתונים
Analytics אתר מוכן לחיבור GA4 / Cloudflare
קישורים פעילים -- קישורי שיתוף שעדיין קיימים ב-KV
נוצרו ב-24 שעות -- שיתופים חדשים מהיממה האחרונה
נוצרו ב-7 ימים -- חלון פעילות זהה לאורך חיי הקישור
פתיחות קישורים -- נספר מרגע העדכון הנוכחי של ה-Worker

שיתופים אחרונים

תצוגה ניהולית ללא פרטי תיק מלאים כברירת מחדל.

מזהה סוג נוצר תפוגה פתיחות סיכום פעולה
אין נתונים להצגה עדיין

סטטיסטיקות אתר

כניסות ומקורות תנועה מיועד לחיבור GA4 או Cloudflare Web Analytics.
פעולות במעבדה בניית תיק, הדפסה, השוואה ושיתוף.
שימוש לפי קטגוריה גמל, השתלמות, פנסיה, פוליסות וחיסכון לכל ילד.
