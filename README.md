# GemulHub — השוואת קופות גמל, פנסיה וקרנות השתלמות

## סקירה כללית
כלי חינמי, מבוסס RTL-עברית, להשוואת קופות גמל, קרנות השתלמות, גמל להשקעה, חיסכון לכל ילד ופנסיה — נתונים ישירות ממשרד האוצר.

---

## קבצי הפרויקט
| קובץ | תיאור |
|---|---|
| `index.html` | דף ראשי (RTL) עם hero, tabs, השוואה עם sidebar ימני |
| `fund.html` | דף פרטני לכל קופה (היסטוריה, מתחרים, מדדים) |
| `js/config.js` | הגדרות API, רשימת יצרנים, קטגוריות ומסלולים |
| `js/api.js` | שאילתות API, cache, סינון, ארגון נתונים |
| `js/app.js` | לוגיקת UI: טאבים, סינון, מיון, טבלאות, חיפוש |
| `css/style.css` | עיצוב ראשי (RTL, CSS variables) |
| `css/fund.css` | עיצוב דף פרטי קופה |

## תכונות שהושלמו

### דף ראשי (index.html)
- ✅ Header יחיד עם ניווט
- ✅ Sidebar סינון בצד ימין (בתוך compare-section)

- ✅ כותרות קטגוריה קליקביליות → עוברות להשוואה
- ✅ סינון: רק קרנות עם תקופת דיווח אחרונה

### דף השוואה
- ✅ Sidebar סינון בצד ימין
- ✅ פילטר אוכלוסיית יעד בתחתית ה-sidebar
- ✅ ללא אפשרות מיון ברירת מחדל
- ✅ מיון עמודות: חודשי, 1Y, 3Y, 5Y, מניות%, חו"ל%, מט"ח% (גבוה→נמוך)
- ✅ ריווח בין קוביות
- ✅ שמות קרנות קליקביליים → fund.html
- ✅ trackList לגמל להשקעה: ללא מסלולי חיסכון לכל ילד/הלכה/סיכון

- ✅ עמודת "מצטברת 1Y" במקום "מתחילת שנה" (YTD)
- ✅ ממוצע קבוצה כולל 1Y

### דף קופה (fund.html)
- ✅ Hero גדול ומעוצב: שם חברה גדול (2rem), מסלול, קטגוריה, מספר קופה, תקופה
- ✅ 5 כרטיסי סטטיסטיקה: חודשי, 1Y מצטברת, 3Y מצטברת, 5Y מצטברת, סך נכסים
- ✅ גרף היסטוריה עם Chart.js (חודשי / 3Y / 5Y)
- ✅ טבלת השוואה למתחרים: קופה מוצגת ראשונה עם ★, כולל עמודת 1Y
- ✅ רשימת מתחרים מסוננת לפי תקופת דיווח אחרונה (זהה לדף ההשוואה)
- ✅ מדדים נוספים: מומנטום (1Y vs. מתחרים, חיובי⬆ / שלילי⬇), עקביות, דירוג קבוצה, חשיפות

## API
- **מקור נתונים:** https://data.gov.il/api/3/action/datastore_search
- **Resource ID:** `a30dcbea-a1d2-482c-ae29-8f781f5025fb`
- **שדות מרכזיים:** `FUND_ID`, `FUND_NAME`, `FUND_CLASSIFICATION`, `SUB_SPECIALIZATION`, `REPORT_PERIOD`, `MONTHLY_YIELD`, `YIELD_TRAILING_1_YR`, `YIELD_TRAILING_3_YRS`, `YIELD_TRAILING_5_YRS`, `TOTAL_ASSETS`, `STOCK_MARKET_EXPOSURE`, `FOREIGN_EXPOSURE`, `FOREIGN_CURRENCY_EXPOSURE`
